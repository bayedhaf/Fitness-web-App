
# Fitness Tracker Web App 🏋️‍♂️

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](https://github.com/yourusername/fitness-web-app/pulls)

A modern web application for tracking workouts, nutrition, and fitness goals with data visualization.



## Features ✨

- **Workout Tracking**: Log exercises with sets/reps/weight
- **Nutrition Dashboard**: Food diary with calorie tracking
- **Progress Analytics**: Visual charts for weight, strength, etc.
- **Exercise Library**: 210+ exercises with instructions
- **Responsive Design**: Works on mobile, tablet & desktop
- **User Authentication**: Secure signup/login with JWT

## Tech Stack 💻

**Frontend**: 
- React.js (Vite)
- Redux Toolkit
- Tailwind CSS
- React Hook Form

**Backend**:
- PHP
- MongoDB (Mongoose)
- JWT Authentication
- REST API
